# patcherly-connector-packages

Distribution repo for Patcherly connector packages. This repository has **no source code** — it only hosts a GitHub Release whose assets are built and uploaded by CI from the main Patcherly app private repo.

## What’s in the release

The release **`connector-packages`** is updated automatically and contains:

- **php-agent.zip** — PHP connector (for universal installer / web installer)
- **node-agent.tar.gz** — Node.js connector
- **python-agent.tar.gz** — Python connector
- **wp-patcherly.zip** — WordPress plugin

- Ignore the other files
